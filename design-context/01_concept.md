# Developer Progression Portfolio

This is not a traditional portfolio.

It is an interactive gamified experience where the user scrolls to progress through the developer’s career.

## Core idea

- Scroll = experience gain
- Experience = XP system
- XP = level progression
- Level = unlocks skills and milestones

The interface behaves like a game HUD.

## Main UI concept

A fixed Profile HUD card displays:

- Name
- Role
- Level
- XP bar
- Unlocked skills, and level increments when repeating skills, creating a dynamic skill badge system that visually represents the developer's expertise and experience in each skill.

The rest of the page is a scroll-driven timeline of milestones.
