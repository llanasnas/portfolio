# UI States

## Level 1 (Beginner state)

- Minimal skills visible
- Low XP bar
- Few milestones unlocked
- Simple HUD layout

## Level 3 (Mid progression)

- More skills visible
- XP bar partially filled
- Milestones expanding
- HUD becomes richer visually

## Level 6 (High level)

- Full skill set visible
- XP bar nearly maxed
- Multiple unlocked milestones
- HUD becomes visually dense and premium

## Level-up state

- Screen glow pulse
- XP bar animates rapidly
- Level indicator increases
- New skills unlock animation
