# Visual System

## Style direction

- Dark mode (deep navy / black base)
- Futuristic SaaS dashboard aesthetic, rounded corners, smooth gradients
- 3D-inspired UI elements with subtle depth
- Subtle glow accents (blue / purple)
- Glassmorphism panels
- Clean typography (Inter / JetBrains Mono)

## UI components style

- Cards: glass + soft border + blur
- Buttons: minimal, glowing hover
- Progress bars: smooth gradient fill
- Icons: consistent simple icon set (Simple Icons / Lucide style)

## Avoid

- Hacker modern terminal style IA inspiration
- Over-glitch effects
- Heavy neon saturation
